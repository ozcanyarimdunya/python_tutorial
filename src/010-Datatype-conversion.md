# Datatype conversion

### Int to float
```python3
>>> i = 10
>>> float(i)
10.0
```

### Float to int
```python3
>>> f = 14.66
>>> int(f)
14
```

### String to int
```python3
>>> s = "123"
>>> int(s)
123
```

### Number to string
```python3
>>> i = 100
>>> str(i)
"100"
>>> f = 1.3
str(f)
'1.3'
```

### Rounding numbers **IMPORTANT**
```python3
>>> i = 23.97312
>>> round(i, 2)
23.97
```
